# Mermaid Lens 验收清单

> 每次发布前依次检查 [[01-简单图表]]、[[02-中等复杂度图表]] 和 [[03-大型图表]]。打开开发者工具（`Ctrl+Shift+I`）并确认 Console 中没有异常。

## 基础渲染

- [ ] 阅读视图中的所有图表均正常渲染
- [ ] 实时预览中的所有图表均正常渲染
- [ ] 图表宽度没有超出笔记正文
- [ ] 相同图表的箭头、渐变和 marker 没有互相串用

## 全局配置

- [ ] 修改 `themeVariables.primaryColor` 后图表重绘
- [ ] 修改 `sequence.actorMargin` 后时序图重绘
- [ ] 输入非法 JSON 时出现提示，并继续使用上一份配置
- [ ] 点击“恢复默认配置”后恢复默认主题
- [ ] 重启 Obsidian 后配置仍然保留

建议临时测试配置：

```json
{
  "theme": "base",
  "themeVariables": {
    "primaryColor": "#DCFCE7",
    "primaryBorderColor": "#16A34A",
    "lineColor": "#15803D"
  },
  "sequence": {
    "useMaxWidth": true,
    "actorMargin": 60,
    "messageMargin": 40
  }
}
```

## 打开 Viewer

- [ ] 单击模式只响应单击
- [ ] 双击模式只响应双击
- [ ] 仅按钮模式不会响应图表正文点击
- [ ] 隐藏展开按钮后按钮立即消失
- [ ] 点击 Mermaid 中的链接不会打开 Viewer
- [ ] 重复切换设置不会产生多个展开按钮

## 平移与缩放

- [ ] Viewer 初次打开时图表居中并适配窗口
- [ ] 鼠标滚轮以光标位置为中心缩放
- [ ] 鼠标左键可以拖拽平移
- [ ] `+`、`=`、`-` 快捷键正常
- [ ] 工具栏放大、缩小、适配按钮正常
- [ ] 双击 Viewer 恢复适配
- [ ] 调整窗口大小后行为正常
- [ ] 触摸设备可单指平移、双指缩放
- [ ] Esc 正常关闭 Viewer

## 生命周期与多视图

- [ ] 左右分屏打开同一笔记时均正常
- [ ] 快速切换笔记后展开按钮正常出现
- [ ] 应用配置后光标与滚动位置不变
- [ ] 禁用插件后按钮和事件被清理
- [ ] 再次启用插件后没有重复按钮或重复触发
